# Contributing

欢迎提交问题和适配修复。站点页面会持续变化，选择器更新应尽量小而可验证。

## 开发流程

1. 使用 Node.js 22 或更新版本。
2. 运行 `npm ci`。
3. 修改前先阅读 `docs/adapter-maintenance.md`。
4. 运行 `npm run check`。
5. 在 Chrome 和 Edge 至少以“仅填入”模式人工验证受影响站点。

## 约束

- 不添加远程脚本、Cookie 读取、私有 API 或遥测。
- 不用单一按钮文字作为唯一选择器，优先语义属性、编辑器类型和稳定结构。
- 不降低发送结果验证标准。
- `submission_unknown` 不得自动改投。
- 新权限必须说明必要性，并优先使用可选权限。
- 不提交个人会话 URL、浏览器资料或单词历史。

提交选择器修复时，请附页面状态说明和对应 DOM fixture 测试，不要附带私人聊天内容。
